### 通过命令行查看可用的翻译服务

您可以通过在命令行中打印帮助信息来确认可用的翻译服务及其使用方法。

```bash
pdf2zh_next -h
```

在帮助信息的末尾，您可以查看不同翻译服务的详细信息。

### 翻译引擎支持政策

#### Tier 1 (官方支持)

**Tier 1 翻译引擎**是当前仓库计划在公开发布流程中优先保持可用的一组引擎。这些引擎的问题应优先在当前仓库中反馈。

目前支持的 Tier 1 翻译引擎包括：
1. OpenAI
2. AliyunDashScope
3. DeepSeek
4. SiliconFlow
5. Zhipu
6. OpenAICompatible

补充说明：

- `SiliconFlowFree` 免费在线接口在本定制分支中仍然保留，管理员打开设置入口后可在 GUI 中选择；
- 但它更适合作为“免费试用/临时使用”方案，不建议视为稳定 SLA 服务。

#### Tier 2（社区支持）

**Tier 2 翻译引擎**属于尽力而为的社区支持范围。
当这些引擎遇到问题时，是否修复将取决于当前仓库的贡献者投入情况与发布优先级。

所有程序支持但未明确列在 Tier 1 下的引擎均被视为 Tier 2 翻译引擎。

#### 已弃用的引擎

以下翻译引擎已被**弃用**，将不再维护或支持：

1. Bing
2. Google
