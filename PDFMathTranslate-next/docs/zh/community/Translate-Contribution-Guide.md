# 文档翻译改进建议

感谢您对本项目的关注！非英文文档由大语言模型自动翻译，因此错误在所难免。如果您在熟悉的语言中发现问题，我们热忱欢迎您提出改进建议。

在开始贡献之前，请花点时间阅读以下指南，以确保您的建议能够顺利被采纳。

# 提交改进建议

当前仓库暂未默认配置自己的 Weblate 工作区。现阶段请通过以下方式提交文档翻译修正建议：

1. 提交 Issue，并明确写出需要修改的文案。
2. 如果你已经知道要改哪个文件，也可以直接提交 Pull Request。
3. 如果你后续为此仓库启用了独立的 Weblate 或其他翻译平台，请把本指南改成指向你自己的工作区，而不是上游项目页面。

提交翻译问题时，建议同时提供：

- 文件路径
- 原始句子
- 建议替换文本
- 是否需要同步更新中英文文档
