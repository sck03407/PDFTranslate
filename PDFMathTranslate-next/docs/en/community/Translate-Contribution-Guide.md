# Suggesting Improvements to the Documentation Translations

Thank you for your interest in this project! The non‑English documentation is automatically translated by large language models, so mistakes are inevitable. If you notice issues in a language you are familiar with, we warmly welcome your suggestions for improvement.

Before you start contributing, please take a moment to read the following guide to help ensure your suggestions can be smoothly accepted.

# Submit Suggestions

This repository does not currently assume a Weblate workspace of its own. For now, please submit documentation translation fixes through one of these channels:

1. Open an Issue and clearly describe the wording that should be changed.
2. Open a Pull Request directly if you already know which file to edit.
3. If you later enable a dedicated translation platform such as Weblate, update this guide to point to your own workspace instead of any upstream project page.

When reporting a translation issue, please include:

- the file path
- the original sentence
- the suggested replacement
- whether the wording should be updated in both English and Chinese docs
